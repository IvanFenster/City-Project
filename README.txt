Проект «Симуяция города»

Данный проект – симуляция города, которая состоит из реальных кварталов испанского города Барселоны, дорог и машин, передвигающимся по ним. Машины не сталкиваются между собой, но иногда встают в затор, из которого их эвакуирует прилетевший вертолет. Пожалуйста, чтобы увидеть всю полноту симуляции, дождитесь затора из машин и эвакуации их вертолетом. Также по городу ездит две машины такси, перемещающиеся между определенными точками на карте, обозначаемыми желтыми маркерами.

Схему классов и файлов можно найти в файле scheme.png. Там же указано, где были использованы паттерны проектирования, а именно singletone, адаптер и фасад.

Чтобы запустить симуляцию города, запустите файл main.py
